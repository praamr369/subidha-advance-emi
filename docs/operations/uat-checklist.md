# UAT Checklist by Module

Use this for final user acceptance testing in a controlled environment.

## Core EMI and payment

- Customer create and customer import preview
- Product create/edit and product import preview/post
- Batch create and batch validation
- Subscription create
- EMI schedule visibility
- Cashier collection with branch/counter context
- Payment reversal visibility

## Direct sale and billing

- Direct sale draft
- Delivery-gated invoice posting
- Receipt creation
- Billing register and billing detail drill-down

## Inventory and manufacturing

- Opening stock preview/post
- Stock overview and stock ledger
- Stock adjustment approval/posting
- BOM create/update
- Production job release, issue, receipt, and completion

## Procurement and finance ops

- Vendor import/create
- Purchase bill draft/post
- Expense voucher
- Accounting books and trial balance

## Workforce and payroll

- Staff import/create
- Attendance entry
- Leave request
- Salary sheet create/post
- Salary payment
- Expense claim and reimbursement
- Staff ledger

## CRM and service desk

- Lead register and follow-up note
- Party detail timeline
- Complaint register
- Return/service ticket
- Service-case detail and linked note flow

## Branch and reporting

- Branch master
- Counter master
- Cashier scoped visibility
- Branch reporting overview

UAT completes only when each module above has at least one successful end-to-end run and any failures are documented with a decision.
