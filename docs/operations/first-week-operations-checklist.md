# First-Week Operations Checklist

Use this during the first live week after rollout.

## Daily checks

- Review branch reporting for every active branch.
- Review cashier payments for missing branch/counter context.
- Review delivery queue and inventory bridge outcomes.
- Review service desk returns, complaints, and credit/debit-note follow-up.
- Review salary/expense/vendor documents only if those modules are already live.

## End-of-day checks

- Counter close checklist completed
- Branch day-close checklist completed
- One finance book review completed
- One inventory summary review completed

## Escalate immediately if

- Cashier cannot collect due to missing branch/counter assignment
- Payment, invoice, stock, or journal entries appear without source trace
- Branch reporting shows amounts against the wrong branch
- Inventory moves without a known service-layer workflow
- Import files are being edited and reposted without validation/preview
