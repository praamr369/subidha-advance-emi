# Vendor Onboarding and Purchase Workflow

1. Create vendor in accounting vendor register (admin only).
2. Optionally create vendor agreement in inventory procurement workspace.
3. Raise purchase request for shortage or sourcing demand.
4. Create purchase order from approved request/vendor selection.
5. Record goods receipt after physical verification.
6. Post goods receipt to increase stock ledger.
7. Create and post vendor bill to move amount into payable.
8. Post vendor payment to settle payable.
9. If return required, create/post purchase return from posted purchase bill.
