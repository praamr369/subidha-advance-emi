# Branch Day-Close and Finance Reconciliation Checklist

## Branch operations close

1. Review the branch reporting overview.
2. Confirm collections by method: cash, bank/card, and UPI.
3. Confirm direct sale totals and active contract posture.
4. Confirm overdue EMI count for the branch.
5. Confirm stock summary and any material branch-specific mismatch.

## Finance review

- Cash book reviewed
- Bank book reviewed
- UPI book reviewed
- Any payment reversal reviewed
- Any credit/debit note posted today reviewed

## Operational review

- Deliveries completed/returned reviewed
- Service desk return and complaint queue reviewed
- Purchase inward and manufacturing movements reviewed when the branch uses them
- Payroll/expense reimbursements reviewed if branch-local processing happened

## Escalation rules

- Do not “fix” mismatches by editing source rows directly.
- Use the operational correction flow for the affected module.
- Record the incident and keep the source trail intact.
