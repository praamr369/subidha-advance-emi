# Architecture Overview

SUBIDHA ADVANCE EMI follows a separated backend and frontend architecture.

## Backend
- Django + DRF
- PostgreSQL
- JWT authentication
- Business-rule driven service logic

## Frontend
- Next.js App Router
- TypeScript
- Tailwind CSS
- Role-based dashboard workflows

## Main design priorities
- additive changes only
- financial integrity
- auditability
- operational simplicity
- future rent/lease compatibility
